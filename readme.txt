== Description ==

**Allows you** to create a wordpress business directory

Supported languages : English	(.pot not provided yet)

Tested with Firefox3, Internet Explorer 7, Safari 3.1

== Installation ==
1. Extract it inside your plugins directory.

2. Create a new page and paste the following code
`<!--wp-ttisbdir-->`

3. Create your first category and add your first listing 

That's it ... Have fun
